Purpose

The Integration Hub is a highly ﬂexible platform that liberates data from across the various systems, augments it, and makes it available for consumption internally, or externally, with no vendor dependency, channeled through the latest available digital touch points. It is an API-enabled Platform, which brings to life an ecosystem delivering a true omnichannel banking experience.

Integration Hub integrates with any existing banking systems. Once the integration is complete, you will be able to:

- Manage operational business rules and workﬂows - Retrieve analytics about transaction - Have full control over your data

Thanks to the Integration Hub, electronic information can be exchanged with an exhaustive array of existing legacy systems, external data providers, credit reference agencies, brokers and lenders.

Rapid Integration

Integration Hub comes with a built-in Decision Engine that is based on BPMN and DMN standards. Using a web modeler business users can respond to market events quickly from anywhere in the world. Revised models can be deployed with no downtime to customers.

Based on business needs whole decisioning process can be automated saving time and resources. Processes can be composed to auto approve/reject applications or send for manual review.

CMS

Business website and content is face of the business and it represents what business is all about. And this content may change frequently. With traditional systems it’s always difficult to tie CMS to back-end transactional applications like Decision Engine. We have created a hybrid solution, Integration Hub extracts all this front matter and deploys it to CDN (Content Delivery Network) guaranteeing fastest response from anywhere in the world.

Since entire front matter is served as static content, basic issues that come OS bundled static servers are eliminated by nature.

Front End also comes with a head-less CMS, that is based on MarkDown. Using friendly and intutive interface business users can create/modify content with no fancy tech knowledge.

Cloud native & Zero downtime

Integration Hub is cloud native. It takes advantage of cutting edge technology and comes pre-configured with features like Load Balancer, Redundancy and Auto scaling.

This technically means application should never go down and in the event of a failure, a new instance spins up and joins the cluster immediately.

Any new features can be deployed with zero downtime.
